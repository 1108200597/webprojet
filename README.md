# bood_donation_project
Description:

The Blood Donation Project is a web-based application designed to connect blood donors with recipients efficiently and securely. The system allows users to register as donors, search for available blood types, and request donations based on location and availability. It aims to simplify the process of finding compatible donors, promote regular blood donations, and support hospitals and organizations in managing blood inventories.

The platform includes features such as donor registration, recipient requests, blood type filtering, and contact management. It also raises awareness about the importance of blood donation and encourages community participation through a user-friendly interface.

 Technologies Used:

Frontend: HTML, CSS, Sass, JavaScript

Backend: PHP, Laravel

Database: MySQL

Additional Tools: GitHub for version control

 Objectives:

Facilitate easy communication between donors and recipients.

Maintain an organized record of blood donors and donations.

Encourage voluntary blood donation and community support.

Ensure quick access to vital information during emergencies.
